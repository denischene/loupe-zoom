🎨 Colore Texte — Réglages

Pour modifier le raccourci clavier, allez dans
about:addons → ⚙️ → Gérer les raccourcis d'extension

⌨️ Commandes au clavier

  • Ctrl+M pour activer/désactiver l'extension. Une fois activée, un petit indicateur apparaît au-dessus du focus.
  • Entrée pour coloriser l'élément focusé. Quand la coloration est active, Entrée sur un élément activable (hyperlien, bouton) valide l'élément.
  • Échap. pour désactiver la coloration.
  • Tab / Maj.+Tab pour déplacer le focus d'élément en élément.

🖱️ Commandes à la souris

  • Cliquer sur l'icône Colore Texte pour activer/désactiver l'extension.
  • Échap. pour désactiver la coloration. Elle reste disponible, un 1er clic la réactive.
  • Un second clic depuis la coloration visible valide l'élément (hyperlien, bouton).
