⌨️ Mode Clavier

  1. Ctrl+Maj+Z pour activer/désactiver l'extension loupe. Une fois activée, un petit logo loupe apparaît alors au-dessus du focus.
  2. Entrée pour faire apparaître la loupe. Quand la loupe est lancée la touche Entrée sur un élément activable (hyperlien, bouton) valide l'élément.
  3. Échap. pour faire disparaître la loupe.
  4. Tab / Maj.+Tab pour déplacer le focus d'élément en élément.

🖱️ Mode Souris

  1. Cliquer sur l'icône Loupe dans la barre d'outils pour activer/désactiver la loupe.
  2. Clic droit pour faire disparaître la loupe. Elle reste disponible au bout de la souris. Il suffit de faire un 1er clic pour la faire réapparaître.
  3. Un 2e clic depuis la loupe visible valide l'élément (hyperlien, bouton…)
  4. Ctrl+molette haut-bas permet de changer de niveau de zoom.

⌨️ Keyboard mode

  1. Ctrl+Shift+Z to toggle the magnifier extension. Once on, a small magnifier logo appears above the focus.
  2. Enter to show the magnifier. When the magnifier is on, pressing Enter on an activable element (link, button) activates it.
  3. Esc to hide the magnifier.
  4. Tab / Shift+Tab to move the focus from element to element.

🖱️ Mouse mode

  1. Click the Loupe icon in the toolbar to turn the magnifier on/off.
  2. Right-click to hide the magnifier. It stays available at the mouse tip. A first click brings it back.
  3. A 2nd click from the visible magnifier activates the element (link, button…).
  4. Ctrl+wheel up/down changes the zoom level.
